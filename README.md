
# TripVillas project

This a collaborative project done within a 4 members team

## API Reference

#### Get all items

```http
  GET https://api-0231.herokuapp.com/results
```

#### Get item

```http
  GET https://api-0231.herokuapp.com/results/${id}
```
#### Add a property
``` http
    POST https://api-0231.herokuapp.com/form
```
